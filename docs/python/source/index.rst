What is Veloxx?
---------------

Veloxx is a lightweight, high-performance data processing library that provides:

* **Fast DataFrame Operations**: Efficient data manipulation with zero-copy operations where possible
* **Memory Safety**: Built in Rust for guaranteed memory safety without garbage collection overhead
* **Cross-Platform**: Works on Windows, macOS, and Linux
* **Multi-Language Support**: Native bindings for Python and WebAssembly/JavaScript

Key Features
------------

* **High Performance**: Optimized Rust core with minimal Python overhead
* **Intuitive API**: Familiar DataFrame and Series operations similar to pandas
* **Type Safety**: Strong typing with automatic type inference
* **JSON/CSV Support**: Efficient parsing and serialization
* **Filtering & Aggregation**: Advanced data filtering and grouping operations
* **Column Operations**: Easy column manipulation, renaming, and selection
